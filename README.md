# Programmers.day

A modern, responsive website dedicated to celebrating **Programmer's Day**, the 256th day of the year.

Programmer's Day is celebrated on the **256th day** of each year (2${^8}$).
-   **Common Years**: September 13th
-   **Leap Years**: September 12th

This project serves as a tribute to the architects of the digital world, featuring a live countdown and interesting facts about the holiday.

## üöÄ Usage

This project is a static site built with vanilla HTML, CSS, and JavaScript. No build step or database is required.

### Local Development
1.  Clone the repository.
2.  Open `index.html` in your web browser.

### Deployment
This site is designed to be hosted on **GitHub Pages**.
1.  Go to your repository settings on GitHub.
2.  Navigate to the "Pages" section.
3.  Select the `main` branch as the source.
4.  Save, and your site will be live!
